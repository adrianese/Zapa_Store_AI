# Checklist de pruebas admin: Subastas y Productos

## 1. Productos
- [ ] Crear producto con talles y stock
- [ ] Editar producto (nombre, precio, talles)
- [ ] Eliminar producto
- [ ] Verificar disponibilidad y stock por talle
- [ ] Subir imágenes y atributos personalizados

## 2. Subastas
- [ ] Crear subasta con uno o más productos
- [ ] Editar subasta (fechas, productos, reglas)
- [ ] Pausar y reanudar subasta
- [ ] Eliminar subasta
- [ ] Listar productos de una subasta
- [ ] Agregar/quitar productos de una subasta
- [ ] Sincronizar subasta con blockchain

## 3. Seguridad y roles
- [ ] Probar acceso solo admin a endpoints críticos
- [ ] Validar protección de rutas y acciones

## 4. Integración
- [ ] Asociar productos a subastas y verificar integridad
- [ ] Verificar que los cambios se reflejan en frontend
- [ ] Probar flujos desde panel admin (UI)

---

Marca cada ítem al completar y reporta cualquier bug o inconsistencia para iterar rápido.
